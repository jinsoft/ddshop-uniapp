# Shopro使用说明

1. 首次安装需要 npm i 加载相关依赖 

2. 基于uni-app，需要下载hbuilderX进行编译

3. 新添加页面进入不了问题：

    把page.json中新加的页面，meta中的auth和async设置为false。重新运行下，一般可以解决。
